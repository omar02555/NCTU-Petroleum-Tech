Submenu Fix Implementation Reference

This page serves as a reference for implementing the submenu fix across all pages of the NCTU Petroleum Tech website.

Note: The submenu visibility issue has been fixed by applying a custom CSS file that changes the background color of dropdown menus to black and text color to white for better visibility.

Implementation Methods

Method 1: Direct CSS Inclusion

Add the following line in the <head> section of each HTML file:

<link rel="stylesheet" href="wp-content/custom-menu-fix.css" type="text/css" media="all">

For pages in subdirectories, adjust the path accordingly:

<link rel="stylesheet" href="../wp-content/custom-menu-fix.css" type="text/css" media="all">

Method 2: JavaScript Loader

Include the JavaScript loader in the <head> section of each HTML file:

<script src="wp-content/custom-menu-fix-loader.js"></script>

This script will automatically check if the CSS is already loaded and add it if necessary.

Method 3: WordPress Theme Integration

If you have access to the WordPress theme files, you can add the CSS to the theme's style.css file or include it via functions.php.

Files Created

  • wp-content/custom-menu-fix.css - The main CSS file with all the fixes
  • wp-content/custom-menu-fix-loader.js - JavaScript to dynamically load the CSS
  • wp-content/themes/nisoz/style-submenu-fix.css - Theme-specific version of the fix
